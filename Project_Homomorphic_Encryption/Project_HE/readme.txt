Nella cartella principale c'è il file sorgente "enc_openssl.cpp" che utilizza la libreria OpenSSL mentre nella cartella My_Palisade c'è il file sorgente "enc_palisade.cpp" che utilizza la libreria Palisade

------------Installazione Palisade-------------
Installare la libreria Palisade seguendo questa guida: https://gitlab.com/palisade/palisade-release/-/wikis/Build-instructions
Dopo aver installato Palisade, seguire questa guida per installare il proprio progetto che usa la libreria Palisade: https://gitlab.com/palisade/palisade-release/-/wikis/Instructions-for-building-user-projects-that-use-PALISADE
